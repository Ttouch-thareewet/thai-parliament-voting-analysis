# thai-parliament-voting-analysis
This project analyzes voting behavior in the Thai parliament using data from Parliament Watch.
The analysis applies Principal Component Analysis (PCA) to parliamentary voting records to uncover patterns in how Members of Parliament (MPs) align politically.

The goal is to explore whether voting behavior reflects clear political blocs, ideological similarities, or deviations from party positions.

In particular, this project investigates:
  1.Political alignment among MPs
  2.Cross-party voting behavior
  3.Potential “cobra” MPs (members who vote against their party line)
